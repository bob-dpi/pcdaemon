```
============================================================

axo2

The Axelsys MachXO2 board has no user visible buttons
or LEDs.  The LEDs that are on the board display the
pin values for the first two peripherals.

The list of driver ID is available as the 'drivlist' 
resource/


RESOURCES
drivlist : list of driver IDs


EXAMPLES
 pcget axo2 drivlist


```
